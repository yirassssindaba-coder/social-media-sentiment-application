## ğŸ§© Troubleshooting

- **Aplikasi tidak bisa jalan di Mac/Linux** â†’ WPF hanya untuk Windows.
- **.NET SDK belum ada** â†’ install .NET 8 SDK, lalu jalankan ulang.
